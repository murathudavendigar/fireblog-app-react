### Built with

- React Components, Props, States and Hooks
- API GET,POST,DELETE and UPDATE requests
- Axios
- React-Router
- Context API
- Toastify
- JSX Elements
- CSS custom properties
- TailwindCSS
- Firebase Auth
- Firebase Database
